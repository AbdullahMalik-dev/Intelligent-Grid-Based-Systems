**By:** Abdullah Malik
**Course:** AP Computer Science A
**Date:** June 2026

## Project Overview
This is my Semester 2 final project for AP Computer Science A. It is a Java-based intelligent system that operates on a 2D grid environment. The program generates a random maze using recursive backtracking, and then an autonomous AI solves it by finding the shortest path to the exit using Breadth-First Search (BFS).

## Features Included
* **2D Array Grid:** The maze is stored and manipulated as a 2D integer array.
* **Recursion:** Uses recursive backtracking to procedurally generate a unique maze every time.
* **Searching Algorithm (BFS):** Implements Breadth-First Search to find the shortest, most optimal path from the start to the exit.
* **Intelligent AI Behavior:** An agent automatically navigates the grid following the path generated by the BFS algorithm.
* **Sorting (Bubble Sort):** Includes a leaderboard system that stores completion times and sorts them using Bubble Sort.
* **Persistence:** Users can save their current maze state to a text file and load it back later.
* **Java Swing GUI:** Fully interactive graphical interface with dynamic visual updates.

## File Structure
* `Main.java` - Contains the Swing GUI, buttons, and graphics rendering.
* `MazeGrid.java` - Manages the core 2D array data structure.
* `MazeGenerator.java` - Handles the recursive maze generation logic.
* `PathFinder.java` - Contains the BFS algorithm and Node class for searching.
* `AIPlayer.java` - Controls the autonomous movement of the AI entity.
* `SaveManager.java` - Handles reading from and writing to `maze_save.txt`.
* `ScoreManager.java` - Manages the leaderboard array and bubble sort logic.

## Setup and Execution Instructions
This project requires no external libraries. It uses standard Java and Swing.

1. Clone or download this repository to your local machine.
2. Open your terminal or command prompt and navigate to the folder containing the .java files.
3. Compile all files by running the following command:
   javac *.java
   java Main
